    Rendering #

    Propósito #

    Realizar indagación teórica-práctica de rendering.

    Tareas #

    1. Realizar una indagación teórica de algún algoritmo de visibilidad o algún método de iluminación global.
    2. Implementar la visualización de un algoritmo de rasterización, e.g., líneas, círculos, antialiasing. Acá se pueden apreciar visualizaciones para la rasterización de un triángulo que pueden servir de modelo.

    Entrega #

    Plazo para hacer commit del repositorio a github: 13/12/21 y 6/12/21 a las 24h, grupos 1 y 2 respectivamente.

    Protocolo presentaciones #

    Exposiciones a ser realizadas la clase siguiente al push de 8m: 6m presentación: discusión del reporte subido a su página + 2m preguntas.
